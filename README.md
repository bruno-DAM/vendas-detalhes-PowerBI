# vendas-detalhes-PowerBI
Projeto desenvolvido no Power BI para uma análise detalhada e interativa de vendas, demonstrando a venda de produtos ao longo de meses e anos, juntamente com o lucro e os segmentos.

### Características

- **Botão de limpeza de filtros**: Permite ao usuário resetar todos os filtros aplicados.
- **Botões para navegar entre páginas**: Facilita a transição entre diferentes seções do relatório.
- **Botões para retornar à Home Page**: Permite que o usuário volte rapidamente à página principal.

## Estrutura do Relatório

### Página Principal

A página principal é projetada para fornecer uma visão geral das vendas, com foco em:

- **Vendas por segmento**: Visualização que destaca as vendas de diferentes segmentos.
- **Vendas por produto**: Análise detalhada das vendas baseadas nos produtos oferecidos.
- **Vendas por país**: Comparação das vendas em diferentes países.
- **Indicadores de desempenho**:
  - Total de Vendas
  - Unidades Vendidas
  - Média de Vendas
  - Soma de Descontos
  - Soma de COGS (Custo das Mercadorias Vendidas)

### Página de Detalhes

A página de detalhes oferece uma análise mais profunda com foco em:

- **Unidades vendidas por produto**: Detalhes sobre a quantidade vendida de cada produto.
- **Histograma de unidades vendidas**: Visualização da distribuição das unidades vendidas.
- **Vendas por mês**: Gráfico que mostra a tendência das vendas ao longo dos meses.
- **Matrix de lucro por ano em cada trimestre**: Análise do lucro obtido, segmentado por ano e trimestre.

### Como Usar

1. **Interagir com os Botões**: Use (com CTRL) os botões de navegação para explorar diferentes seções do painel.
2. **Aplicar Filtros**: Utilize o botão (com CTRL) de limpeza de filtros caso precise reiniciar suas análises.
3. **Explorar Dados**: Analise as visualizações em cada página para entender melhor os padrões de venda.
